This project is a fully functional Python application that automates attendance tracking for campus or community events. It uses structured data, loops, functions, and error handling to create a reliable system that records sign-ins, removes entries, displays attendance, and saves a professional report. The goal is to demonstrate how programming supports real-world service-learning needs by improving organization and reducing human error.
